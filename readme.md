# 🎮 משחק כרטיסיות ללימוד אנגלית

פרויקט זה הוא משחק דפדפן פשוט שנועד לעזור לילדים דוברי עברית ללמוד אוצר מילים בסיסי באנגלית. הוא נבנה ככלי לימודי אישי עבור ילד בן 8, עם דגש על מילים פשוטות, צלילי אותיות ניקוד (vowels), ותרגום ברור בעברית.

## ✨ תכונות עיקריות

* **כרטיסיות אינטראקטיביות:** לחץ על כרטיסייה כדי להפוך אותה ולראות את התרגום בעברית ואת התמונה.
* **מנוע קול (TTS):** כפתור "Speak" משתמש במנוע ה-TTS המובנה של הדפדפן כדי להקריא את המילה באנגלית.
* **קטגוריות דינמיות:** טעינה קלה של אוספי מילים שונים (חיות, אוכל וכו') מתוך תפריט בחירה.
* **הרחבה פשוטה:** הוספת מילים חדשות או קטגוריות שלמות דורשת עריכה של קבצי JSON בלבד, ללא צורך בשינוי קוד ה-JavaScript.
* **Fallback לתמונות:** במקרה שתמונה חסרה או שהנתיב אליה שבור, המשחק יסתיר את שטח התמונה ויציג במקומו את התרגום בעברית בפונט גדול יותר.

## 🚀 איך להשתמש (למשחק)

אין צורך בהתקנה!
פשוט פתח את הקובץ `index.html` בכל דפדפן מודרני (כמו כרום, פיירפוקס או ספארי).

## 🛠️ איך להוסיף תוכן (למפתחים)

הפרויקט בנוי בצורה מודולרית כדי לאפשר הוספת תוכן בקלות.

### 1. הוספת מילים חדשות לקטגוריה קיימת

1.  גש לתיקייה `data/`.
2.  פתח את קובץ ה-JSON של הקטגוריה הרלוונטית (למשל, `data/animals.json`).
3.  הוסף אובייקט JSON חדש למערך, לפי הפורמט הבא:

    ```json
    {
      "word": "lion",
      "hebrew": "אריה",
      "image": "images/animals/lion.png",
      "vowel": "i"
    }
    ```

4.  הוסף את קובץ התמונה (למשל `lion.png`) לתיקייה המתאימה (למשל `images/animals/`).

### 2. הוספת קטגוריה חדשה (למשל "Food")

זהו תהליך של 3 שלבים:

1.  **יצירת קובץ הנתונים:**
    * צור קובץ JSON חדש בתיקייה `data/` (למשל, `data/food.json`).
    * מלא אותו במערך של מילים, באותו פורמט כמו בדוגמה למעלה.

2.  **עדכון האינדקס הראשי:**
    * פתח את הקובץ `categories.json` (שנמצא בתיקייה הראשית).
    * הוסף אובייקט חדש למערך, המקשר לקובץ החדש שיצרת:

        ```json
        [
          {
            "name": "Animals",
            "id": "animals",
            "file": "data/animals.json"
          },
          {
            "name": "Food",
            "id": "food",
            "file": "data/food.json"
          }
        ]
        ```

3.  **הוספת תמונות:**
    * צור תיקייה חדשה עבור התמונות של הקטגוריה (למשל `images/food/`).
    * הוסף את קבצי התמונות שלך לשם.

### 3. הורדת תמונות אוטומטית (כלי פיתוח)

כדי להאיץ את תהליך הוספת התמונות, קיים סקריפט Node.js בשם `fetch_images.js`.

**מה הוא עושה?**
הסקריפט קורא את כל הקטגוריות והמילים מקבצי ה-JSON שלך, בודק אילו תמונות חסרות, ומנסה להוריד אותן אוטומטית מ-Unsplash (מקור לתמונות חינמיות).

**איך להפעיל:**

1.  ודא שיש לך [Node.js](https://nodejs.org/) מותקן במחשב.
2.  הפעל את הטרמינל שלך ונווט לתיקיית הפרויקט.
3.  אם זו הפעם הראשונה, הפעל `npm install` (למרות שבסקריפט הנוכחי אין תלויות חיצוניות).
4.  הפעל את הסקריפט:

    ```bash
    node fetch_images.js
    ```

הסקריפט יבדוק את כל המילים וידלג על אלו שכבר יש להן תמונה.

## 📂 מבנה הפרויקט

```
readenglish/
├── data/
│   ├── animals.json         # אוצר מילים: חיות
│   └── ...                  # (עוד קבצי נתונים)
├── images/
│   ├── animals/
│   │   ├── cat.png
│   │   └── ...              # (עוד תמונות של חיות)
│   └── ...                  # (עוד תיקיות לקטגוריות אחרות)
├── app.js                   # הלוגיקה המרכזית של המשחק (JavaScript)
├── categories.json          # קובץ אינדקס ראשי של כל הקטגוריות
├── fetch_images.js          # (כלי פיתוח) סקריפט Node.js להורדת תמונות
├── index.html               # שלד האפליקציה (HTML)
├── README.md                # הקובץ הזה
└── style.css                # עיצוב האפליקציה (CSS)
```

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Audio:** Web Speech API (SpeechSynthesisUtterance)
* **Development:** Node.js (עבור סקריפט הורדת התמונות)
